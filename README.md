# Mobile eCommerce Clothing Store App

A modern and responsive mobile eCommerce clothing store application built with React.

## Features

- Product listings and categories
- Product details page
- Shopping cart functionality
- Responsive design for mobile and desktop devices
- User-friendly interface
- Smooth navigation experience

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3

## Installation

Clone the repository:

bash
git clone https://github.com/Tiirow/Mobile-eCommerce-Clothing-Store-App.git


Install dependencies:

bash
npm install


Run the application:

bash
npm start
